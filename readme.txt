User dashboard notifications plugin lets administrators create notifications to be displayed in admin panel for users or group of users(roles).

== Description ==

Add and mange notifications from Administrator account to notify other registered users in the site, registered users will dismiss the notification and it will be reflected in admin notification list.



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/user-dashboard-notifications` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the admin "User Dashboard niotfications" Item menu to add and list notifications
